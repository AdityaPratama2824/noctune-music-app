# Noctune Music
Welcome to Noctune Music, a modern music discovery platform built to explore songs, artists, playlists, and night vibes through a clean and immersive experience.

# About Project
Noctune is a modern fullstack music discovery web application built using the MERN stack. Users can search for songs and artists, listen to music previews, save favorites, create playlists, and explore trending music through a clean and modern interface.

## Features
* Register & Login
* Search Songs & Artists
* Music Preview Player
* Favorite Songs
* Playlist Management
* Trending Music
* Recent Searches
* Responsive Design
* Dark Mode UI

## Tech Stack
* **Database** : MongoDB
* **Backend** : ExpressJS
* **Frontend** : ReactJS
* **Runtime** : NodeJS

## Installation & Setup
```bash
# Clone Repository
git clone https://github.com/your-username/noctune-music-app.git

# Enter The Project Folder
cd noctune-music-app

# Frontend Setup
cd frontend
npm install
npm run dev

# Backend Setup
cd backend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside the `backend` folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
